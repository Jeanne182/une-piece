# Une Piece
A temple run based game made by IMAC

### Build : 
Here are the bash commands to build the project :

```bash
mkdir build && cd ./build
cmake ../src
make
```

Execute project :
`./UP/<built_file>`

Exemple: `./UP/UP_SDLtemplate`
